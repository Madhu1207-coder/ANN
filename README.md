## 🚀 Overview

This project demonstrates how to predict stock market prices using **Long Short-Term Memory (LSTM)**, a type of Artificial Neural Network (ANN) designed to work with time-series data. It includes steps such as preprocessing, model building, training, and evaluating a model on historical stock data.

---

## 🌟 Features

- 📈 Data loading and visualization
- 🧹 Data normalization and reshaping
- 🧠 LSTM-based time-series prediction
- 🔁 Model improvement with a better dataset
- 📊 Performance visualization and error analysis

---

## 📦 Dataset Used

- Source: Stock price data (CSV format)
- Content: Date-wise historical prices of a listed stock
- Features: Open, High, Low, Close, Volume

---

## 🔁 LSTM Flow

1. 📥 Load Dataset  
2. 📊 Visualize historical trends  
3. 🧹 Normalize & reshape data  
4. 🧠 Train LSTM model  
5. 🔍 Predict and visualize results  
6. 🧪 Test with improved dataset
